四、关键操作：Clawdbot对接微信
Clawdbot官方没有直接的微信通道，但是咱可以通过
Matrix插件+微信机器人
实现联动，全程零成本。👍
1. 安装Matrix插件（Clawdbot端）
Matrix的网站是matrix.org。国内需要科学上网。
# 安装Matrix插件
clawdbot plugin install clawdbot-plugin-matrix
# 重启Clawdbot生效
clawdbot restart
然后在Clawdbot的Web界面（http://localhost:18789）进入「拓展」-「Matrix」，注册一个Matrix账号，完成绑定就行～
2. 部署微信机器人（对接Matrix）
咱用开源的微信机器人实现微信和Matrix的消息互通，这样Clawdbot就能通过Matrix接收微信指令，执行操作后再把结果返回到微信，相当于直接接管微信～
# 克隆微信机器人项目（最新版）
git clone https://github.com/wechaty/wechaty.git
cd wechaty
# 安装依赖
npm install
# 配置Matrix对接（修改配置文件）
vim config.js
# 替换以下配置（填你的Matrix账号和Clawdbot地址）
module.exports = {
matrix: {
host: "http://localhost:18789",
user: "你的Matrix账号",
pass: "你的Matrix密码"
},
wechat: {
autoReply: true,
port: 3000
}
}
# 启动微信机器人
npm start
启动后会弹出二维码，用微信扫码登录，就完成微信和Clawdbot的对接了！

六、避坑指南：新手必看的3个常见问题
咱踩过的坑，不让大家再踩！整理了3个最常见的问题，附解决方案👇
问题1
：执行clawdbot命令提示“command not found”？
解决方案：重启终端，或者执行
source ~/.bashrc
刷新环境变量；
问题2
：微信机器人扫码后提示“登录失败”？
解决方案：确保微信是最新版，关闭微信的设备保护，重新扫码；
问题3
：Clawdbot能接收指令但不执行？
解决方案：检查端口18789是否被占用，执行
clawdbot restart
重启服务，确保Matrix插件绑定成功。
