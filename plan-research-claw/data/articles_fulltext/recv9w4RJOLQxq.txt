## Clawdbot 安装前置条件：下载 Node.js
（1）去

https://nodejs.org/en/download

（2）

（3）点击

## Clawdbot 安装步骤
#### 1、创建 Clawdbot 存放地址
新建一个你用于放置
Clawdbot
的文件夹，并右键复制文件地址（Mac电脑长按Option键右键进行复制）
#### 2、快捷安装 Clawdbot
（1）打开电脑终端输入
“cd”
，按空格键，然后粘贴刚才复制的地址
cd
刚才复制的地址
（2）按
"enter/return"
键确认后，依次输入以下内容
// 全局安装最新版本的 Clawdbot
npm install -g clawdbot
@latest
// 启动clawdbot项目，并依次选项进行配置
clawdbot onboard --install-daemon
#### 3、配置 Clawdbot 内容
接下来一步一步跟着主包的引导进行即可：
（1）选择
Quick Start

（2）选择

#### 4、Telegram对接方式
（1）在
Telegram

BotFather

（2）进入对话后，输入
/newbot

英文

（3）复制 BotFather 发你信息里面的
HTTP API

Clawdbot

#### 7、重启网关和验证
关闭终端后重新打开，然后输入
// 重启网关
clawdbot gateway restart
// 验证
clawdbot channels status --probe
提示 reachable 则说明配置成功了
#### 8、启动 clawdbot
tui模式
// 开启网关
clawdbot gateway
// 启动 Tui 模式打开clawdbot
clawdbot tui
WebUI 模式
// 启动 WebUI 模式
clawdbot gateway
--port
18789
--verbose
在浏览器中输入
127.0.0.1:18789
打开web页面
