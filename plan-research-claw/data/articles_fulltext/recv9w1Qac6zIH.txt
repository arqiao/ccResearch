面向个人开发者：MoltBot接入火山方舟Coding Plan配置流程
火山方舟 Coding Plan 支持在 Moltbot 中使用，需完成以下操作：
访问火山方舟 Coding Plan 新用户特惠活动，按需订阅套餐。套餐介绍参见套餐概览
https://www.volcengine.com/docs/82379/1925114?lang=zh
。
在开通管理页面
https://console.volcengine.com/ark/region:ark+cn-beijing/openManagement?LLM=%7B%7D&advancedActiveKey=subscribe
选择或切换目标模型，无需在工具中额外变更模型配置。
支持最新的 Doubao-Seed-Code、DeepSeek-V3.2、GLM-4.7、Kimi-K2 模型，也可开启 Auto 模式，火山方舟将根据场景选择最优模型。
在 Coding Plan 场景下，查看和修改配置的入口和上述一致，其中需要修改的核心配置信息如下：
baseUrl：
https://ark.cn-beijing.volces.com/api/coding/v3
apiKey：获取API Key（链接：
https://console.volcengine.com/ark/region:ark+cn-beijing/apikey
）
models：ark-code-latest
修改主要模型配置
"model"
: {
"primary"
:
"doubao/ark-code-latest"
},
"models"
: {
"doubao/ark-code-latest"
: {
"alias"
:
"doubao"
}
}
配置模型供应商
"models"
: {
"providers"
: {
"doubao"
: {
"baseUrl"
:
"https://ark.cn-beijing.volces.com/api/coding/v3"
,
"apiKey"
:
"YOUR_API_KEY"
,
"api"
:
"openai-completions"
}
}
}

面向企业客户：MoltBot接入方舟模型配置流程
👉 第一步：查看当前配置
方式一：在 MoltBot UI 中查看
在 Web UI 中，选择
Settings-Config-Authentication
，在底部选择
Raw
方式查看配置信息。
方式二：在终端查看
在终端执行以下命令查看配置信息。
cat ~/.clawd/config.json
👉 第二步：修改配置接入方舟模型
MoltBot 支持快速接入火山方舟上的豆包, DeepSeek, GLM, Kimi等模型。
下面以
豆包1.8模型
为例，讲解接入流程。（如果是想接入 deepseek、glm 等其他模型，替换成方舟官网上对应的 base_url、model 即可）
1.修改主要模型配置
在 agents 参数下，找到 model 和 models，改为您要接入的模型名称
"model"
: {
"primary"
:
"doubao/doubao-seed-1-8-251228"
},
"models"
: {
"doubao/doubao-seed-1-8-251228"
: {
"alias"
:
"doubao"
}
}
2.配置模型供应商
在 models 参数下，修改您要接入的 base_url 以及 API KEY（可在方舟 API 接入中找到）。
"models"
: {
"providers"
: {
"doubao"
: {
"baseUrl"
:
"https://ark.cn-beijing.volces.com/api/v3"
,
"apiKey"
:
"YOUR_API_KEY"
,
"api"
:
"openai-completions"
}
}
}
3.修改认证配置
在 auth 参数下，修改认证配置
"auth"
: {
"profiles"
: {
"doubao:default"
: {
"provider"
:
"doubao"
,
"mode"
:
"api_key"
}
}
}
👉 第三步：保存配置并重启服务
在 MoltBot 配置界面完成修改后，先单击
Save
，保存完成后单击
Update
。
值得注意是，MoltBolt 提供了强大的个性化 AI Agent 体验，但用户需注意安全与隐私风险。
由于工具可访问设备所有数据、数字账户并通过多端口联网，存在一定安全隐患。建议在专用设备而非含敏感信息的设备上部署 MoltBolt，定期检查并调整权限设置，并为火山方舟模型 API 密钥设置访问限制。
