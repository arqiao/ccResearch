第一步：开启WSL2并安装Ubuntu
按下开始键，搜索 “PowerShell”。
右键点击 “Windows PowerShell”→“以管理员身份运行”。
运行以下命令，使用默认的 Ubuntu 发行版安装 WSL：
wsl --install
如果系统提示需要重启，请重启电脑。
重启后，Windows 会自动打开一个 Ubuntu 窗口以完成设置；请为 Linux 选择用户名和密码（这与你的 Windows 密码不同）
第二步：打开 Ubuntu（WSL2）并进行更新
按下开始 → 输入“Ubuntu” → 打开 Ubuntu 应用。
在 Ubuntu 终端中运行：sudo apt update && sudo apt upgrade -y
出现提示时输入你的 Ubuntu 密码。这个命令会确保你的 Linux 环境是最新的。

安装完成后可使用node -v命令来检查版本号，若显示版本号为22.X或更高，说明安装正常。
第四步：在WSL2中安装Clawdbot
继续在WSL终端中运行以下命令：
sudo npm install-g clawdbot@latest

继续在WSL终端中运行以下命令：
clawdbot --help
如果能正常打印出帮助内容，则说明Clawdbot安装成功！
第五步：运行指引并安装守护进程
指引程序会连接你的模型（Claude/OpenAI 等）并设置网关服务，以便机器人能够 24 小时不间断运行。
继续在WSL终端中运行以下命令：
clawdbot onboard --install-daemon

第六步：检查Clawdbot的运行状态
先进行一下基本健康检查：
clawdbot doctor
如果在检查结果中发现有些问题（Issues）,你可以通过
clawdbot doctor --fix
这个命令来修复。
接下来检查网关状态：
clawdbot status
如果在检查中发现问题，它会尝试自动修复。
然后，我们可以使用以下命令打开Web可视化仪表盘，用于监控Clawdbot的运行状态、设置频道、实例、会话期、Skills等。
clawdbot dashboard

接下来我们用Clawdbot 完成第一个简单的工作：每日总结。
在开始之前，我们先用 Clawdbot doctor 检查运行状态，这个命令请大家牢记，并养成习惯，以便Clawdbot能够顺畅地为你工作。
现在，请你保持这个WSL终端窗口一直处于打开状态。然后我们打开第二个终端窗口。需要注意的是，不要打开powershell或command命令行窗口，而是在“开始”中选择Ubantu图标进入新的窗口。
下面，请在新窗口中输入以下命令：
clawdbot agent \
--agent main \
--message "请担任我的私人助理。把今天当作一个普通的工作日，给我一个简单的、包含时间的 5 项待办事项清单，其中包括一项健康任务和一项学习任务。" \
--thinking medium
稍等片刻，Clawdbot就会根据你的要求为你制订一份<日工作计划>:
