飞书接入 OpenClaw
这个开源项目是为最近那个火出圈的 OpenClaw 做的一个
飞书连接器
。
OpenClaw 虽然能让 AI 操控电脑、跑代码，但它官方支持的聊天软件大多是 Discord 或者 Telegram 这种国人用得少的。
这个开源项目就是为了填这个坑，
让你的 OpenClaw 能直接住进飞书里。
只要在电脑上装好 OpenClaw ，填上飞书开放平台的机器人参数，你就能
在手机飞书上远程指挥家里的电脑干活了。
不管是让它查资料、监控服务器状态，还是跑个自动化脚本，发条消息它就能执行，不用专门为了跟 AI 对话再去开个别的软件。
安装方式：
openclaw
plugins install
@m1heng
-clawd/feishu
然后在飞书开放平台创建自建应用，并在凭证页面获取 App ID 和 App Secret，然后开启所需权限、配置事件订阅。
具体可以参考开源项目主页的内容。
开源地址：https://github.com/m1heng/clawdbot-feishu
等你全部搞好，
群里 @它就行
，能识别图片和文件。
你可以直接发个 PDF 文档、Excel 表格或者一张截图给它，它能直接读取内容并进行处理。回复时支持富文本卡片，代码块会有高亮，表格也能正常显示，看起来很舒服。
支持 WebSocket 长连接模式。你不需要折腾内网穿透或者有公网 IP，
只要家里的电脑能上网，飞书就能连上它。
除了发文字，它也能反过来
发送文件和图片给你
。配合 OpenClaw 本身的能力，你甚至可以用飞书远程命令它生成一个报表文件发回给你，或者发一条 /new 指令来重置上下文开启新话题。
02
Awesome OpenClaw Skills
简单来说，这就是目前大火的开源 AI Agent 项目 OpenClaw 的 Skill 库。
OpenClaw 本身是一个能跑在本地、真正接管电脑干活的 AI 智能体，而
这个仓库收集了几百种 Skill，专门用来给它扩充能力
，让它不仅仅是个陪聊的机器人。
这里面的东西非常实用，你能在里面找到各种现成的脚本和配置，
让 AI 学会操作浏览器抓数据、管理本地文件、自动发邮件，甚至是写代码和部署服务。
这个项目由 VoltAgent 团队维护，整理得挺全，分类也清晰。
无论你是想用 AI 来处理日常杂事，还是想搞点复杂的开发工作流，去这里淘一淘现成的轮子绝对比自己从头造要快得多。
对于正在折腾 OpenClaw 的玩家来说，这基本上是一个
必看的资源站。
开源地址：https://github.com/VoltAgent/awesome-openclaw-skills
03
MoltWorker
这个是 Cloudflare 推出的一个开源项目，教你
如何在 Cloudflare Workers 的 Sandbox 容器环境中部署 OpenClaw 个人 AI 助手。
这个开源项目构建了一套完整的无服务器架构，利用 Cloudflare Access 进行安全身份验证，并通过 R2 对象存储来实现配置和对话历史的持久化，从而避免了维护传统 VPS 服务器的繁琐。
在功能上，它不仅支持通过 Web 界面管理，还能接入 Telegram、Discord 等聊天平台，甚至内置了基于 Chrome DevTools Protocol 的浏览器自动化能力，
适合开发者快速搭建低成本的私有 AI Agent。

OpenClaw 的记忆层
memU 是一个给长期运行的 AI Agent 用的记忆层，比如 OpenClaw。
它会在后台悄悄把对话、文档、行为日志这些东西
整理成结构化记忆
，一点点积累，帮应用在不同会话之间保留用户的目标、偏好和上下文，不用每次都从头问起。
对开发者来说，就是在现有代理外面加了一层专门管记忆的基础设施，既能减少反复喂上下文的次数，也能
降低长期的 LLM 调用成本。
它内部把记忆分成资源、条目和类别，像
一个分层的个人知识库。
从落地场景看，它更适合做一直在
旁边观察和学习那种助手
：长期跟踪你的阅读和搜索习惯，主动推送可能感兴趣的内容。
帮你记住邮件风格、常用回复和重要联系人，自动归类和起草邮件，也能盯着投资偏好和市场事件，在触发条件满足时给提醒或建议。
简单说，如果你在做任何需要记得住人和事的智能体产品，memU
可以把记忆这块的活先干好，让上层的逻辑专心做交互和决策。
开源地址：https://github.com/NevaMind-AI/memU
