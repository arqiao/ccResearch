一、 先填上篇巨坑！插件安装报错+微信风控，一次性根治
上篇好多宝子反馈2个核心问题，先优先解决，避坑第一步，踩过的坑咱绝不回头😎
坑1： 安装Matrix插件报错spawn npm ENOENT（Windows专属）
核心原因：Node.js没配环境变量，OpenClaw找不到npm命令，跟插件没关系！
✅ 进阶根治方案（一步到位，亲测有效）
1.  先卸载旧版openclaw：npm uninstall -g openclaw
2.  重新装Node.js22+，必须勾选Add to PATH，装完重启电脑（划重点！重启才生效）
3.  管理员终端执行2条命令，全局重装+授权，杜绝权限问题
npm install -g openclaw@latest --registry=https://registry.npmmirror.com
openclaw upgrade --force
4.  再装插件：openclaw plugins install clawdbot-plugin-matrix，秒成功！
坑2： 微信扫码风控登不上，网页版直接被拦
核心原因：网页版协议被腾讯风控，新号必拦，老号不稳！
✅ 进阶方案：弃用网页版，换WeChatFerry本地Hook方案（但也是减少风控概率。零风控可以使用微信企业版接口，后面会专门写一篇文章。）
不用改复杂配置，直接替换上篇微信机器人，3步搞定，新号老号都能用👇
# 1.  克隆稳定版WeChatFerry项目（国内源，下载超快）
git clone https://gitee.com/lich0821/WeChatFerry.git
# 2.  进入项目装依赖，国内镜像提速，不报错
cd WeChatFerry/WeChatFerry.CLI
npm install --registry=https://registry.npmmirror.com
# 3.  启动+对接Clawdbot，直接扫码登录
npm run start -- --matrix-host http://localhost:18789
二、 进阶核心1： 给AI助理加记忆！记住你的习惯，越用越懂你
基础版Clawdbot用完就忘，进阶版直接开持久化记忆，让它记住你的文件路径、常用指令、工作习惯，不用反复交代，比助理还贴心🥰
2步开启记忆功能，配置即生效
1.  打开Clawdbot网页端（http://localhost:18789），点击左侧「设置」-「记忆与上下文」
2.  关键配置拉满，直接抄作业，附代码级配置（怕错就直接复制）
// 本地修改配置文件，持久化记忆不丢失（路径：C:\Users\你的用户名\.openclaw\config.json）
{
"memory": {
"enable": true, // 开启记忆
"persistence": true, // 持久化保存，重启不丢
"contextLength": 20, // 记忆最近20条对话
"customMemoryPath": "C:\\Users\\Lenovo\\.openclaw\\memory.json" // 记忆文件路径，自定义
}
}

- 第二次直接发：帮我生成周报并存好

五、 进阶避坑指南：3个高频问题，秒解决不踩雷
进阶玩法配置多，3个坑提前避开，少走冤枉路👇
1.  插件装完不生效？重启Clawdbot就好，命令：openclaw restart，插件必须重启加载；
2.  定时任务不执行？检查系统时间同步，指令时间用24小时制，比如下午5点写17:00；

六、 终极进阶： 外网不通？国内服务器替代方案，无需代理
好多宝子说Matrix官网打不开，不用慌！直接弃用外网Matrix，用国内方案实现微信-Clawdbot互通，全程无需代理，稳定到离谱～
✅ 国内替代方案：用HTTP接口直接对接，2行命令改配置，完美适配
# 1.  关闭Matrix插件，启用HTTP接口
openclaw plugins disable clawdbot-plugin-matrix
openclaw plugins enable clawdbot-plugin-http
# 2.  微信机器人对接HTTP接口，直接互通
npm run start -- --http-host http://localhost:18789 --http-port 8080
配置完直接用，功能完全一样，国内网络丝滑不卡顿，小白也能搞定～
