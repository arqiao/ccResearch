Clawdbot 太火了，昨天写了怎么对接飞书的，结果一堆人来问怎么接钉钉，这篇就来具体聊聊。
这样的配置步骤有点多，这一期我加入了视频演示，方便大家看得更清楚。
现在 Clawdbot 其实已经改名叫 Moltbot 了，但方法其实都一样。
开始之前需要准备什么？
一台电脑、一个Qwen 账号（ 这地址注册：
https://chat.qwen.ai/）
当然，还需要钉钉账号，个人的即可。配置的过程需要加入一个公司，你随便起个公司名加入就好了。
仅此而已，不需要买 Mac Mini，也不需要买云主机，更不需要公网 IP 地址。
风险提示
：Clawdbot 需要完全的文件访问权限，而且外面可以直接连到本机，所以会存在较大的风险。如果你的长期使用，建议使用单独的没有敏感数据的电脑。
01｜安装 Clawdbot
比较有意思的是，整个过程安装 Clawdbot 反而是最简单的，看这个视频就可以了。（

如果你想知道细节，也可以从下面的一步步解释开始。
安装 Clawdbot 先从一条命令开始，在 Mac/Linux 上打开你的终端（Teminal 或者 iTerm 之类的），然后黏贴下面的命令：
curl
-fsSL https://molt.bot/install.sh | bash
如果是 Windows，把这条命令改成：
iwr
-useb https://molt.bot/install.ps1 | iex
除此之外，Windows 后续所有操作和 Mac/Linux 都一样。
请注意：这里的命令不要出现全角、半角空格的问题。

02 配置钉钉
配置钉钉的过程非常简单。
第一步：安装钉钉插件
直接执行下面的命令就好了:
clawdbot plugins install https://github.com/soimy/clawdbot-channel-dingtalk.git
cd
~/.clawdbot/extensions/dingtalk/
npm install zod
第二步：创建应用
到钉钉的开发者后台创建一个应用：https://open-dev.dingtalk.com/
如果你看到的不是下面的界面，直接点顶部的“
应用开发
”。
这里的应用名称和描述，就是机器人的名字，根据你的需要写就好了。
第三步：设置机器人
，这是非常重要的一步。
点击左侧的“
机器人
”，然后打开“
机器人配置
”，输入一些基本信息：
最关键的：
要把消息接收模式改成 Stream 模式，然后点发布。
最后一步
，发布这个应用：
到这里，钉钉机器人应用就创建并发布好了。
