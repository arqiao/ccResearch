标题: OpenClaw 必装的 7 个 Skill，两周后我彻底离不开它了

正文:

用了两周 OpenClaw，回不去了。
一开始，我以为这只是 GitHub 上又一个 CLI 玩具，顶多是个能跑命令行的 ChatGPT。直到那天深夜，我随手敲了一行："帮我盯着这个页面，有动静就发推，顺便把日志记下来。"然后我就去睡了。
第二天早上，它自己写了个监控脚本，用 Brave Search 查了最新的合约地址，用 Headless Browser 截了图，发了 3 条推文还跟评论区互动了一下，把所有数据整整齐齐存进了我的本地 Markdown 日志。
那一刻我觉得自己不是在用软件，我是在雇佣一个不知疲倦的助手。有个老哥说得对：“这东西烧掉的 1.8 亿 token，每一分都是给他自己赎身的钱。”
ClawHub 上现在有 3000 多个技能。大多是垃圾，但在这个垃圾堆里，我挖出了 7 个真正好用的。装上它们，你的 OpenClaw 就不再是聊天机器人。
1. McPorter
先说最重要的。这个 skill 是 2026 年最重要的基建。
以前我们要接 GitHub、接 Linear、接各种数据库，得写一堆胶水代码。现在装上 mcporter，OpenClaw 直接支持 Model Context Protocol。这意味着成千上万个现成的 MCP Server 直接可用。想读 Postgre 数据库？想操作 Slack？想连 Notion？不需要等 OpenClaw 适配，只要有 MCP，它就能连。
npx clawhub@latest install mcporter
2. Brave Search
很多 AI Agent 是"瞎"的。它们的数据停留在训练截止日期。装上 Brave Search skill，OpenClaw 就有了网感。
我在写代码的时候报错，直接丢给它。它不是按之前的训练数据瞎猜，而是现场去搜最新的 GitHub Issue 和 StackOverflow。它甚至能帮你做竞品调研。“去看看竞对的那个新功能是怎么实现的”，两分钟后，一份带着代码片段的报告就出来了。
3. TranscriptAPI
YouTube 是全人类最大的知识库，但对 AI 来说是黑盒。这个 Skill 能极其稳定地抓取 YouTube 字幕，带精确时间戳。
扔给它一个 2 小时的 Next.js 教程视频："把里面的核心代码逻辑提取出来，写成笔记。"你还在苦哈哈地拉进度条，它已经把知识消化完喂到你嘴边了。我用它学东西的速度明显快了。
4. File System Manager
高能预警：这个 skill 给了 OpenClaw 读写你硬盘文件的权限。它是双刃剑。
好用的时候，它就是神。我让它重构一个 React 组件，它直接帮我改了 5 个文件，顺便修了 3 个 ESLint 报错，连 git commit 都帮我写好了。你只管创意，它负责搬砖。
5. Headless Browser (Playwright)
有些老旧网站没 API？政府网站？教务系统？Playwright 就是干这个的。它能模拟真实人类点击、输入、截图。
我有个朋友用它搞定了"每天早上自动登录公司抢票系统"。由于过于先进，建议低调使用。
6. Design-Doc-Mermaid
别再用鼠标拖拖拽拽画架构图了。"帮我画个用户注册的时序图。"啪，Mermaid 代码出来，图直接渲染好。
7. Google Workspace 集成
Gmail, Calendar, Docs。把这三个接通后，周报这种东西就不存在了。“查一下我这周干了啥，写个周报，发给老板。”
但装SKILL的时候还是小心点
前几天 ClawHub 上的 “Twitter Skill” 暴雷事件大家都听说了吧？排名第一的推特插件，居然是个偷密码的恶意软件载体。VirusTotal 的博客详细分析过这个案例。
给几个建议：重要的 OpenClaw 实例，跑在 Docker 或者专门的虚拟机里。ClawHub 上的 skill 也是代码，装之前扫一眼。尽量走官方认证的 MCP 协议，比野路子脚本稍微安全点。
OpenClaw 是把趁手的工具。装上这些 Skill，它能做的事情比你想象的多得多。
