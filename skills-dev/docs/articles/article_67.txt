 id="js_content" style="visibility: hidden; opacity: 0; "> 这段时间全网都在聊这个火遍全网的“小龙虾”： OpenClaw （前身为 Moltbot ，最初名为 Clawdbot ）。 其中有很多用户的反馈是： “东西是好东西，能跑在本地也很酷，但是……除了聊天，我好像还是得手把手教它写代码、配环境，感觉像是在带一个刚毕业的实习生，有点累。” 确实，一个裸奔的 Agent，哪怕模型再强，也只是一个聪明的“空壳”。它需要手，需要脚，需要工具。 如果你最近在玩 OpenClaw 或其他任何一款 Code Agent，大概率已经意识到一件事： Agent 强不强，模型能力虽然很重要，但关键在 Skills。 模型只是大脑，Skills 才是手、是工具、是执行力。 就在 Clawdbot 刚爆火的时候，GitHub 上就有大佬做了一个关于 Clawdbot 的 Skills 技能合集，而且一直在更新，也一直在跟随官方改名。 它就是： awesome-openclaw-skills ，已经收获了 5.3K Star。 它在短时间内迅速收录了 700+ 个实用技能插件，涵盖了从写代码、做 SEO、剪视频到订外卖、管房贷的 30+ 个领域。 最让人兴奋的是，它带来了一种类似 Linux apt 或 Mac brew 的体验——你不需要自己造轮子，只需要一行命令，就能把别人的智慧安装进你的 AI 里。 这个仓库基本可以算是目前 OpenClaw 生态里最完整的一份技能索引库。 GitHub： https://github.com/VoltAgent/awesome-openclaw-skills 主要分类 1、开发 工程向 • 代码分析/自动重构 • 安全审计（漏洞扫描、依赖风险） • 性能分析与优化 • 自动化测试 • DevOps/CI/脚本执行 2、Web 自动化 • Web 开发辅助 • 浏览器自动化 • 爬虫/数据抓取 • SEO 分析 • 表单自动填写 3、AI 多模态 • 图像生成 • 文生图/图生图 • AI 模型集成 • 文本处理/内容生成 4、内容 生产力 • 写推文/写博客 • 笔记管理（Notion、Markdown） • 日程管理 • 邮件处理 • 文档整理 5、生活 泛自动化 • 购物/订外卖 • 理财辅助 • 健康追踪 • 智能家居 • 出行信息查询 当然还有更多的种类，更实用的技能，这里就不一一罗列了。 快速安装指定技能 这个仓库最核心的逻辑就是：像使用 npm 、 apt 或 brew 一样管理 AI 的技能。 一条命令，技能到位： npx clawdhub@latest install <skill-slug> 注：替换为具体技能的标识即可，仓库内每个技能都标注了对应的slug，直接复制使用就行~ 如果是手动安装，就还是安装老样子，将技能文件夹复制到以下位置之一： • 全局： ~/.openclaw/skills/ • 项目： <project>/skills/ 小提醒：优先级规则是「工作区 > 本地 > 内置技能」，避免技能冲突！ 写在最后 对于 OpenClaw 用户来说，这个仓库绝对是“必备收藏”——不用再四处找技能、踩兼容的坑，700+技能随取随用，让原本强大的本地AI助手，战斗力直接翻倍； 对于开源爱好者和开发者来说，这里不仅能找到优质的技能参考，还能参与贡献，和全球的开发者一起完善OpenClaw生态，积累自己的开源经验。 开源的魅力，就在于互相分享、共同成长。这个宝藏仓库，既有实用价值，又有社区温度，让我们一起来解锁本地 AI 的无限可能！ GitHub： https://github.com/VoltAgent/awesome-openclaw-skills 如果本文对您有帮助，也请帮忙点个 赞👍 + 在看 哈！❤️ 在看你就赞赞我！
