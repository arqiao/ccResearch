标题: 无限进化的剧本和分镜？我用skills帮你做到了！！

正文:

在当前的 AI 编程与创作领域，无论是 OpenCode、Cursor 还是各类 Agent 框架，我们正受困于要对所有大大小小的事情做决策，所以，我在想，能不能将剧本、分镜的前期撰写的流程缩短，把重复性的工作交给AI，而skills完美符合我的要求，不仅能承载我的知识经验，还可以自我迭代，无限进化！
我对skills写剧本有三个要求：
1.AI 自己制定标准（基于长期记忆和知识库）；
2.AI 自己执行任务
3.AI 自己审核结果，并根据偏差反向修改标准。
而且可以通过在对话中，让他记忆住你的要求并自动写入skills里，这就是——“自我进化”。（灵感来源于卡兹克老师）
这里面的核心逻辑：自己卷自己
其实原理特简单。 我把 Agent 拆成了两个分身，虽然可能用的是同一个模型，但逻辑上，它们各司其职，形成了一个完美的闭环：
执行者，这就是个老实人。 只负责读取
SKILL.md
，然后埋头苦干，生成内容。
剧本审核师，它作为剧本、分镜大师的经验知识凝练，会根据前者产出的内容进行审核打分，如若不符，则打回重写。
我们首先可以使用阶跃AI帮你检索全网优秀的电影剧本理论，它不仅能帮你阅读书籍，还可以提取视频，并帮你总结为一个优秀的skills。
我直接把 5本世界顶级的编剧理论书籍（包括罗伯特·麦基的《故事》、悉德·菲尔德的《电影剧本写作基础》、约翰·特鲁比的《故事写作大师班》等），连同几十部影史经典电影的剧本文件，一股脑喂给了 Agent。
在这里我用Anitigravity打开了这个skills，你生成好的skills也可以在opencode中打开它。
你能看到我生成了两个核心skills，一个是制作剧本的，一个是剧本审核的，扮演执行和审阅两个角色。
最终，在 Antigravity（或 OpenCode）的文件系统中，它生成了两个核心文档，编剧核心技能和电影剧本创作。
接下来我们把skills放到opencode里让它开始执行。
实战演练——《怪奇物语》互动小说
接下来，我把这套 Skills 扔进 OpenCode 里跑起来，并设定了一个测试场景：基于《怪奇物语》世界观的互动剧本生成。
效果只能用“卧槽”来形容。
它不再是那个只会续写“然后发生了什么”的聊天机器人。它变成了一个能够实时构建剧情的Showrunner（剧集统筹）。 当我输入一个简单的动作（比如“我决定强行过载天线”），它没有立刻给出结果，而是触发了内部的“双核”循环：
执行者写了一段：“Alex 推上了推杆，天线爆炸了，怪物死了。”
审核者（后台日志显示）：“驳回。描写过于平淡，缺乏生理痛感。违反 S 级标准第 3 条‘感官欺骗’。建议增加听觉描写和物理代价。”
执行者又重新生成：“没有任何声音。只有纯粹的震动。那是超越听觉的物理打击……”
这一过程在几秒钟内完成了数次迭代。最终呈现给我的，是一个充满了电影质感、感官细节丰富、逻辑严密的 S 级互动剧本。
最骚的是，它拥有 “长期记忆”。 通过 evolution.json 文件，它记录下了每一次“被驳回”的原因。比如它发现我喜欢“克苏鲁风格的不可名状感”，它会自动修改自己的审核标准，在未来的生成中加大这一权重的比分。 它在陪我玩的过程中，自己变得越来越懂我，也越来越专业。

既然剧本能进化，那视觉语言能不能进化？ 影视工业不仅仅是文字，更是画面。于是，我发出了第二道指令：“根据你现在掌握的剧本技能，去学习并生成一套关于分镜脚本（Storyboard）的 Skills。”
这里有个小坑需要提醒大家：在 OpenCode 或类似的系统中，Skills 不一定非要叫 "Skills"。 只要你让系统将这些规则、模版、知识库写入 长期记忆（通常是项目根目录下的特定 Markdown 或 JSON 文件），并配置好读取协议，它就能成为系统灵魂的一部分。
这里有个小坑，我要提醒大家一下。 Skills 不一定非要叫 Skills，只要让 OpenCode 把它写入 长期记忆 中就行。
需要注意的是，skills不用一定叫skills，让opencode写入长期记忆中即可。
我们能看到它生成了对应的分镜。
在这个实验中，Skills 的真正牛逼之处，不在于它自动写了多少字，而在于它实现了一种 “经验的数字化永生”。
试想一下，一个在该行业摸爬滚打几十年的老编剧、老导演，他脑子里那些只可意会不可言传的经验——对节奏的把控、对光影的敏感、对人性的洞察——在过去，只能通过带徒弟、写书这种极低效率的方式传承。信息的衰减是巨大的。
但现在，我们可以通过通过不断的对话、反馈、修正，将这些长达数十年的经验，凝练为一个 Markdown 文档。
它极大地减少了人与人之间的沟通成本。 以前，你要跟概念设计师讲半天“我要那种赛博朋克但也带点复古的感觉”，对方可能还是画出一坨屎。 现在，你把你的审美标准写入 Skills。Agent 不需要睡觉，不需要喝咖啡，不会有情绪波动。它一天 24 小时为你工作，每一秒都在以你设定的最高标准审视产出。
当一个 Agent 能够自行从原著中提取世界观，自行撰写符合 S 级标准的剧本，自行拆解分镜，甚至（结合现在的视频生成模型）自行生成样片，并且在整个过程中不断自我修正、自我迭代……
影视工业化，这个喊了多少年的口号，也许真的要在硅基生物的手中实现了。 未来的创作者，可能不再是一个个伏案写作的个体，而是一个个 “Agent 架构师”。我们不再直接生产内容，我们生产“生产内容的逻辑”。
